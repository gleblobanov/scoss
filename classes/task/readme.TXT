0. Скопировать плагин в папку admin/tool
1. Загрузить в папку task сертификат и ключ
2. В файл onlineedu_lib.php вставить номера сертификата и ключа. Примерно строчка 12-13
3. В файле onlineedu_lib.php определить константу  CURLDOMENONLINEEDU
4. В файле onlineedu_lib.php определить константу  MAILUSERERRORONLINEEDU
5. Создать таблицу transfer_onlineedu. Код в файле createtable.txt
6. У курса должно быть доп поля
		id_course_onlineedu с идентификатором курса на сцосе. Так определяется по каким курсам отправлять народ
		session_onlineedu - сессия курса на сцос
		transfer_onlineedu - флаг показывающй отправляется ли информация на сцос
7. В таблице линк надо точно знать номер oauth (issuerid=2) (сейчас реализовано автоматическое определение)
